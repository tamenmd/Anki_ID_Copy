# Security Policy

Anki ID Copy is a local Anki add-on. It does not run a server and does not send
your collection or any other data anywhere — it only reads your selected notes
and writes search strings to your clipboard (and, for the Due Siblings feature,
builds a local filtered deck).

## Reporting a vulnerability

Please report security issues privately through GitHub Security Advisories:
<https://github.com/tamenmd/Anki_ID_Copy/security/advisories/new>

For non-sensitive issues, a regular [GitHub issue](https://github.com/tamenmd/Anki_ID_Copy/issues)
is fine. Please include your Anki version and operating system.

## Supported versions

The most recent release receives fixes.
